Tweet your Woocommerce products to Twitter on publish/update automatically.

== Description ==

This is a simple plugin for your woocommerce site to upload/post your products to Twitter automatically. It will do the magic when you publish new product through woocommerce. It will be shared on your Twitter account.

== Installation ==

To work properly, woocommerce should be installed before installing wootweet.

Here are few steps to follow:

1. Upload `wootweet` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Select WooSocio Logins page from wordpress menu and login.
4. That's it. You're ready to go!

== Frequently Asked Questions ==

= How can I get App for Twitter? =

You can get from https://apps.twitter.com/.

= What if I don't want to tweet a product? =

If you uncheck the box 'Tweet it?' in publish area, it will not be tweeted.
